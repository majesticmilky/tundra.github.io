Tundra



Mute

?mute @member time reason

This command will mute the member mentioned for the given time though the mute can be overridden with the unmute command




Unmute

?unmute @member

This command will override the mute command and unmute the mentioned member




Settings

?settings [muteRoleID, memberRoleID, prefix] [For ID's, add the id here, if changing the prefix, put the new prefix here]

This command will update the database for the mute role and member role, or you can change the prefix


DISCLAIMER: The mute role and member role must be declared before using the mute command




Ban

?ban @member reason

This command will ban the mentioned member




Kick

?kick @member reason

This command will kick the mentioned member




Ping

?ping

This command will show you the latency of the bot




Suggestion

?suggestion [suggestion]

This command allows all members to suggest moderation commands!




Slowmode

?slowmode [time] [minutes, seconds, hours]

This command will increase or decrease the slowmode




Clear

?clear [1-100]

This command will bulk delete messages that are within 14 days old
